Oiyo is an enhanced engineering framework built on top of UniApp that replaces repetitive project setup with conventions. It ships with a file-system-based auto-router, a named layout system, app-level shared state via `defineRootContext`, page meta declarations, and navigation guards—all wired together through a single Vite plugin. Whether you're building WeChat mini-programs, Alipay mini-programs, H5, or native apps, Oiyo gives you a consistent, opinionated structure that scales.

## Why Oiyo?

Building UniApp projects traditionally means writing the same scaffolding over and over—manually registering pages, duplicating layout wrappers, passing state through props or Pinia, and repeating navigation logic across dozens of files. Oiyo eliminates this noise.

## Get started

<Steps>
  <Step title="Install Oiyo">
    Add the Oiyo package to a new or existing UniApp project.

    ```bash
    pnpm add @skiyee/oiyo
    ```
  </Step>
  <Step title="Add the Vite plugin">
    Register `OiyoPlugin` in your `vite.config.ts`.

    ```typescript vite.config.ts
    import OiyoPlugin from '@skiyee/oiyo/plugin'
    import { defineConfig } from 'vite'

    export default defineConfig({
      plugins: [OiyoPlugin()],
    })
    ```
  </Step>
  <Step title="Initialize your app">
    Use `createSSRApp` from `@skiyee/oiyo/runtime` in `src/main.ts`.

    ```typescript src/main.ts
    import { createSSRApp } from '@skiyee/oiyo/runtime'
    import App from './App.vue'

    export function createApp() {
      const app = createSSRApp(App)
      return { app }
    }
    ```
  </Step>
  <Step title="Create your first page">
    Add `src/pages/home/index.vue` and declare its meta inline.

    ```vue src/pages/home/index.vue
    <script setup>
    import { ref } from '@skiyee/oiyo/runtime'

    definePageMeta({
      layout: 'default',
      style: { navigationBarTitleText: 'Home' },
    })

    const message = ref('Hello Oiyo!')
    </script>

    <template>
      <view>{{ message }}</view>
    </template>
    ```
  </Step>
</Steps>

<Tip>
  Run `oiyo dev` to start the development server. Oiyo automatically picks up new pages in `src/pages/` without any manual registration.
</Tip>
